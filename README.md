# 林间随笔 · 个人博客

一个纯前端、全功能的个人博客网站，支持文章列表、详情阅读、标签/归档筛选、搜索和本地笔记管理。

## 功能亮点
- 📝 博客文章展示与全文阅读
- 🏷️ 按标签 / 归档筛选文章
- 🔍 全局搜索（标题、摘要、正文）
- 📌 本地笔记管理（添加/删除，数据存于浏览器）
- 📱 手机、电脑自适应布局

## 部署到 GitHub Pages
1. 将本仓库克隆或上传 `index.html` 文件
2. 进入仓库 Settings → Pages → 分支选 `main`，文件夹选 `/ (root)`
3. 保存后等待1分钟，访问 `https://你的用户名.github.io`

## 自定义文章
编辑 `index.html` 中 `const posts = [...]` 数组，按格式修改或增删文章。

## 技术栈
原生 HTML/CSS/JS，无依赖，图标使用 Font Awesome。
